# **App Name**: Nightcap Nosh

## Core Features:

- Product Catalog Browsing: Users can browse a curated selection of liquors and complementary snacks, complete with detailed product descriptions and high-quality images.
- Shopping Cart & Checkout: Allows customers to add items to a secure shopping cart, adjust quantities, and complete purchases through a multi-step checkout supporting various payment options.
- User Authentication & Age Verification: Secure user registration and login powered by Firebase Authentication, including a mandatory age verification step to comply with liquor sales regulations.
- Real-time Order Tracking: Provides users with live updates on their order status from placement to delivery, including driver location, through in-app notifications managed via Firestore.
- Customer Support Chat: Integrated chat feature for immediate assistance with orders or product inquiries, ensuring quick resolution during late-night hours, with chat data stored in Firestore.
- Loyalty & Rewards Program: Rewards frequent customers with exclusive discounts, free items, or special gift hampers after a set number of purchases, with program data managed through Firestore.
- Admin Dashboard: A comprehensive backend interface for managing product inventory (including age-restricted items), orders, user accounts, and delivery logistics, all powered by Firestore.

## Style Guidelines:

- Primary color: Deep Amethyst (#553399), a sophisticated and rich blue-violet, ideal for evoking a premium, late-night feel while maintaining contrast.
- Background color: Dark Plum (#24202A), a heavily desaturated and very dark shade of the primary hue, providing an elegant backdrop that emphasizes the vibrant UI elements.
- Accent color: Vibrant Blue (#5990F2), an analogous hue that is brighter and more saturated than the primary, perfect for drawing attention to calls-to-action and interactive states.
- Headline font: 'Bebas Neue' (sans-serif) for a strong, modern, and impactful presence. Body text and descriptions: 'Open Sans' (sans-serif) for excellent readability and a clean, contemporary feel across all screen sizes. Note: currently only Google Fonts are supported.
- Sleek, minimalist line icons with subtle rounded edges for an approachable and modern aesthetic. Icons for interactive elements may incorporate subtle crimson or gold accents on hover.
- Mobile-first, responsive design with a clean, spacious grid layout emphasizing high-quality product visuals. Features prominent product cards with clear pricing and 'Add to Cart' functionality. The homepage will showcase a bold banner highlighting '24/7 Availability' and featured products for easy late-night navigation.
- Subtle and smooth hover effects on product cards (e.g., slight scale-up, border glow) and buttons to provide engaging feedback. Fluid transitions for page navigation and modal pop-ups, complemented by custom loading animations for an enhanced user experience without perceived delays.
